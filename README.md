## 介绍

本仓库使用安信可ESP8266模组来接入腾讯云物联开发平台，支持微信连连小程序/smartconfig/绑定/控制；

方式一：提供腾讯物联专有AT指令ESP8266定制模组固件接入；

方式二：提供ESP8266 RTOS 环境 腾讯物联专有SDK 二次开发接入；
